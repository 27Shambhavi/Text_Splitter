# RAG Text Splitter Components

A modular collection of **text splitter components for Retrieval-Augmented Generation (RAG) pipelines**.  
This repository focuses on transforming raw documents into **retrieval-optimized chunks** suitable for embeddings, vector databases, and LLM-based question answering.
